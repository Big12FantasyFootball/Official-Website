Handbook

Code 1 · Participation
All members must be active participants in league groupchat(s).
Code 2 · Trash Talk
Permitted if appropriate and harmless. Harmful comments reviewed for disciplinary action.
Code 3 · Injury Exploitation
Exploiting a leaguemate's lack of knowledge of a player's injury during a trade is a fairness violation; can be overruled by lead council.
Penalty → Trade overruled by lead council
Code 4 · Third-Party Influence
While in trade discussions, no third-party source may influence either side based on personal preferences.
Penalty → Waiver wire privileges revoked
Code 5 · Favoritism
No league mate or council member shall show favoritism to any other member.
Penalty → Demotion and/or probation
Code 6 · Trade Finality
All trades are final. Only overruled if proven coercion to detriment of game integrity. No vetos.
Code 1 · Bribery
No member shall accept or decline a trade through bribery.
Penalty → Immediate termination + forfeit buy-in
Code 2 · Collusion
No member shall purposely lose a trade to give another member an unfair advantage.
Penalty → Immediate termination + forfeit buy-in
Code 3 · Waiver Dumping
No member shall purposely drop players to benefit another member.
Penalty → Immediate termination + forfeit buy-in
Code 4 · Presidential Abuse
President must not use tools to affect scores, rosters, or lineups of any member.
Penalty → Impeachment
Code 5 · Throwing
No member shall purposely start underperforming players to benefit their opponent.
Penalty → Probation and/or termination
Code 6 · Investigation
Any cheating accusation requires a full investigation by Internal Affairs before any conclusion is reached.
Code 1 · Selection
All members must complete the full selection process, unless grandfathered in or given Presidential immunity.
Code 2 · Deadlines
Missing deadlines is a voluntary withdrawal, making the applicant ineligible for consideration.
Code 3 · Buy-In
All members must pay the $150 buy-in to the President before the draft. Failure forfeits league spot indefinitely.
Penalty → Forfeiture of league spot
Code 4 · Financial Debt
A member in financial debt to another member for a substantial period will be terminated immediately.
Penalty → Immediate termination
Code 5 · Acknowledgment
All members must pass an oral exam and verbally and physically acknowledge the handbook.
Code 1 · Presidential Authority
The President holds the highest authority over all members, including lead council.
Code 2 · CEO
The CEO is appointed by the President and holds 2nd command for all decisions.
Code 3 · Supervisor
The League Supervisor is appointed by the CEO and forms the third leg of lead council.
Code 4 · Overruling
A Presidential decision can only be overruled if all 11 other members vote against it unanimously.
Code 5 · Discipline
The Head of Internal Affairs handles discipline referred by the Supervisor. All actions require Presidential review before enforcement.
Code 6 · Impeachment
President can only be impeached after proven violation + unanimous 11-member vote.
Requires: Proven violation + 11/11 vote
Code 7 · Council Removal
Council member removed only if President + 8 members (75%) vote in favor.
Requires: President + 8 members
